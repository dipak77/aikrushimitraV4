# AI Krushi Mitra — Rural Accessibility & Usability (WCAG 2.1)

> **Version:** 1.0 | **Status:** Approved | **Owner:** UX Lead  
> **Last Updated:** 2026-06-28

---

## 1. Low-Literacy Support Guidelines

1.  **Icon-Heavy Interface:** Every key action must pair with a distinct, literal vector icon (e.g., a cow for livestock, a leaf for crop diagnosis, market bags for prices).
2.  **Voice-First Interaction:** Provide voice recording capabilities on every core page.
3.  **No Text-Density Blockages:** Keep instructions brief (maximum 2-3 lines of text inside cards) to prevent screen clutter.

---

## 2. Contrast & Screen Scaling Specifications

*   **Outdoor Sunlight Contrast:** Minimum text contrast ratio of **7:1** against backgrounds (surpassing WCAG AAA requirements).
*   **Large Touch Targets:** Tap targets must measure at least **48x48 CSS pixels** with 8px margin spacing to prevent misclicks on budget mobile screens.
*   **Dynamic Font Resizing:** Supports OS-level system font magnification up to **200%** without text clipping or layout overlap.
